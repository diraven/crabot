# Crabot

Is a discord bot that helps with the discord server management.

## Usage

### Development
- `cp example_docker-compose.override.yml docker-compose.override.yml`
- `cp example_.env .env` - adjust config variables to your liking.
- `make run`

### Deployment
- `cp example_.env .env` - adjust config variables to your liking.
- `make run`
